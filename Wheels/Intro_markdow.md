# How to use markdow to doc

## markdown pros 
* easy to learn，focus on content not style
* compatible with all platforms and html 
* can be used to generate pdf, html, etc

## markdown cons
* not as powerful, flexible as html

---
## Common markdow syntax

1. Titles, up to 6 levels
    #1 - level 1     
    ##2 - level 2     
    ###3 - level 3    
    ####4 - level 4   
    #####5 - level 5  
    ######6 - level 6

2. paragraphs
    using the empty line to separate paragraphs
    or using the html <br> tag

3. highlight
   * ** bold**
   * *** bold italic*** 
   * __underline__
   * ___underline italic___
   * ~~strikethrough~~

4. list
    * bullet list
    * 1. number list
    * - 1. mixed list
  
5. link
    * [link text](url)
    * [link text](url "title")
    * ![alt text](url) -- image link
        example: with HTML
        ![alt text](url "title")

6. quote
    * > quote
    * >> quote with list
        
7. code
    * `inline code`
    * ```python 
    ``` -- for code block

8. horizontal rule
    * ---
    * ***
    * ___

9. table
    | header 1 | header 2 |
    | --- | --- |
    | content 1 | content 2 |
    | content 3 | content 4 |

10. LaTex math
    * $inline math$
    * $$
    $$ -- for math block

11. mermaid flow chart 
    * ```mermaid
    ``` -- for flow chart
